JavaScript Object and Methods - Practical Tasks

1. Basic Object Creation

  • Create an object `person` with properties: `name`, `age`, and `city`. Add a method `greet()` that logs: "Hello, my name is [name]."

2. Dynamic Property Assignment

  • Create an empty object `car`. Dynamically add properties `make`, `model`, and `year` using dot and bracket notation.

3. Object Constructor

  • Write a constructor function `Book` that takes `title`, `author`, and `year` as arguments. Create two `Book` objects and log their details.

4. Object Methods - Object.keys()

  • Create an object `student` with properties: `name`, `grade`, and `subject`. Use `Object.keys()` to log all the keys of the object.

5. Object Methods - Object.values()

  • Use `Object.values()` on the `student` object to log all the values.

6. Object Methods - Object.entries()

  • Iterate over the key-value pairs of the `student` object using `Object.entries()` and log them in the format `key: value`.

7. Object.assign() Method

  • Create two objects `source1` and `source2` with some overlapping properties. Merge them into a new object `target` using `Object.assign()` and log the result.

8. Working with Prototypes - Object.create()

  • Create a prototype object `animal` with properties `type` and a method `sound()`. Use `Object.create()` to create a `dog` object and override the `sound` method to log "Bark".

9. Object Freezing and Sealing - Object.freeze()

  • Create an object `settings` with properties `theme` and `fontSize`. Use `Object.freeze()` and try to modify its properties. Log the result.

10. Object Freezing and Sealing - Object.seal()

  • Create an object `profile` with properties `username` and `email`. Use `Object.seal()` and try adding, removing, and updating properties. Log what works and what doesn't.

11. Property Checking - Object.hasOwnProperty()

  • Create an object `game` with properties `title` and `genre`. Check if it has a property `platform` using `hasOwnProperty()`.

12. Dynamic Property Check

  • Write a function `checkProperty(obj, prop)` that takes an object and a property name, and returns whether the property exists in the object.

13. Custom Methods in Objects - Calculator

  • Create an object `calculator` with methods `add`, `subtract`, `multiply`, and `divide`. Each method should take two numbers as arguments and return the result.

14. Custom Methods in Objects - Library System

  • Create a `library` object with a method `addBook(title)` that adds a book to its `books` array, and a `listBooks()` method that logs all books.

15. Iterating Over Objects - for...in Loop

  • Create an object `course` with properties `title`, `duration`, and `instructor`. Use a `for...in` loop to log all properties and their values.

16. Iterating Over Objects - Object.keys() + forEach()

  • Use `Object.keys()` and `forEach()` to iterate over an object `movie` and log all keys and their corresponding values.

17. Nested Objects

  • Create a `classroom` object with properties `name`, `students` (an array of student objects), and a method `addStudent(student)` to add a new student object to the `students` array.

18. Cloning Objects

  • Write a function `cloneObject(obj)` that creates a deep clone of the given object using `Object.assign()` or spread syntax. Test it on a nested object.

19. Object.defineProperty()

  • Create an object `employee` with properties `name` and `position`. Use `Object.defineProperty()` to add a non-writable, non-enumerable property `salary`.

20. CRUD Operations with Objects

  • Create an object `taskManager` with:
    • An array `tasks` to store task objects.
    • Methods: `addTask(task)`, `deleteTask(taskId)`, `updateTask(taskId, updatedTask)`, and `listTasks()`.
