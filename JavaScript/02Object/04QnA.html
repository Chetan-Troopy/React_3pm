JavaScript Object Questions

Basic Questions

Q1: What is an object in JavaScript?
A: An object is a collection of properties, where each property is defined as a key-value pair. For example:
let obj = { name: "John", age: 30 };
Q2: How do you create an object in JavaScript?
A: Objects can be created using object literals, the Object constructor, or Object.create(). Example with an object literal:
let person = { name: "Alice", age: 25 };
Q3: How do you access object properties?
A: Object properties can be accessed using dot notation or bracket notation. Example:
console.log(person.name); // Dot notation
console.log(person["age"]); // Bracket notation
Q4: What is the difference between dot notation and bracket notation?
A: Dot notation is used when the property name is a valid identifier. Bracket notation is used when the property name includes special characters, starts with a number, or is dynamic.
Q5: How do you add a new property to an existing object?
A: You can use dot notation or bracket notation. Example:
let obj = { name: "John" };
obj.age = 30; // Using dot notation
obj["city"] = "New York"; // Using bracket notation

Intermediate Questions

Q1: What is Object.keys() used for?
A: It returns an array of all the keys of an object. Example:
let obj = { name: "John", age: 30 };
console.log(Object.keys(obj)); // ["name", "age"]
Q2: What does Object.values() do?
A: It returns an array of all the values of an object. Example:
console.log(Object.values(obj)); // ["John", 30]
Q3: What is Object.entries()?
A: It returns an array of key-value pairs of an object. Example:
console.log(Object.entries(obj)); // [["name", "John"], ["age", 30]]
Q4: What is Object.assign() used for?
A: It is used to copy or merge properties from one or more source objects into a target object. Example:
let target = {};
let source = { a: 1, b: 2 };
Object.assign(target, source);
console.log(target); // { a: 1, b: 2 }
Q5: What is Object.create()?
A: It creates a new object with a specified prototype object. Example:
let proto = { greet: () => "Hello!" };
let obj = Object.create(proto);
console.log(obj.greet()); // Hello!
