Practical Tasks for Arrays

Basic Tasks

  • Create an array of your favorite fruits and add a new fruit at the end using the push method.
  • Remove the last element from an array of numbers using the pop method and display the removed element.
  • Use the shift method to remove the first element from an array of colors and print the updated array.
  • Add an element at the beginning of an array of cities using the unshift method and print the array.
  • Check if a specific value (e.g., "Banana") exists in an array using the includes method and display the result.

Intermediate Tasks

  • Reverse the order of elements in an array of numbers using the reverse method.
  • Loop through an array of strings using the forEach method and print each element.
  • Use the for...of loop to iterate over an array of objects (e.g., people with names and ages) and log their names.
  • Find the index of a specific element (e.g., "Apple") in an array using the indexOf method.
  • Use lastIndexOf to find the last occurrence of the value 5 in an array of numbers.

String Manipulation Tasks

  • Use the join method to convert an array of words into a single sentence separated by spaces.
  • Split a string into an array, sort the array alphabetically using sort, and join it back into a string.
  • Use the join method with a custom separator (e.g., " - ") on an array of country names.

Sorting Tasks

  • Create an array of random numbers and sort it in ascending order using sort.
  • Sort an array of strings alphabetically using the sort method.
  • Sort an array of objects (e.g., people with names and ages) by their age using the sort method.

Array Manipulation Tasks

  • Add multiple elements to an array using push in a single operation and display the updated array.
  • Remove multiple elements from an array by chaining pop calls.
  • Use unshift to add two elements at the beginning of an array and print the result.
  • Create a function that checks if an array includes a specific value and returns "Found" or "Not Found".

Combination of Methods

  • Reverse an array, then use join to combine the elements into a single string.
  • Use forEach to create a new array that contains the length of each string in an array of words.
  • Find the index of a value in an array using indexOf, remove it using splice, and display the updated array.
  • Combine push, pop, shift, and unshift to demonstrate adding and removing elements at both ends of an array.

Advanced Tasks

  • Use for...of to iterate over an array of arrays (2D array) and print each inner array.
  • Use forEach to create a new array by doubling the values of all numbers in an existing array.
  • Write a function that takes an array and a target value, then uses indexOf to find its position or return "Not Found" if it's not present.
  • Create an array of strings and sort them in descending order using sort and a custom comparator function.
  • Create a function that takes an array and reverses its elements without using the reverse method.
  • Write a function that finds the most frequent element in an array using a combination of forEach and lastIndexOf.

Instructions for Students

  • Solve each task in a separate code block.
  • Write comments to explain each step in your solution.
  • Test each method with different data to observe how it behaves.
